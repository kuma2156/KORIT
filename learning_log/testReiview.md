그럼 얘가 수학으로 연산이 안된다
이걸 Java 특정 클래스의 특정 메서드는 저런식으로 String 숫자 형태를 int로 바꿔주는 메서드가 있다
그건 시험문제

```java
public static void main(String[] args) {
    String age= "123";
    int intAge = Integer.parseInt(age);

    System.out.println(intAge - 100);
}
```
